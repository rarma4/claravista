Navegação
de usuários

A ClaraVista é uma ferramenta de
captura de dados de navegação de
usuários. A partir da instalação de
uma apenas uma linha de código em
um site ou e-commerce capturamos
todos os eventos de todos os
usuários. Com a identificação de cada
usuário que se registra por
e-mail, criamos perfis individuais.
